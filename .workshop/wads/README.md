# WAD Files

## freedoom1.wad

This is a free, open-source game data file from the **Freedoom** project.

- **Website**: https://freedoom.github.io/
- **License**: BSD License
- **Source**: https://github.com/freedoom/freedoom

Freedoom is a complete replacement for the original Doom IWAD files, allowing you to play Doom-compatible games and mods without needing the commercial game data.
